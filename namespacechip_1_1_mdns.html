◆ ChipMdnsPublishService()

CHIP_ERROR chip::Mdns::ChipMdnsPublishService ( const MdnsService *  service)

This function publishes an service via mDNS.

Calling the function again with the same service name, type, protocol, interface and port but different text will update the text published. This function will NOT take the ownership of service->mTextEntries memory.

Parameters
[in]serviceThe service entry.
Return values
CHIP_NO_ERRORThe publish succeeds.
CHIP_ERROR_INVALID_ARGUMENTThe service is nullptr.
Errorcode The publish fails.
